# GuitarTabTool

A lightweight **guitar tablature tool** designed to help guitarists and developers work with tabs and MIDI playback in a simple workflow.

GuitarTabTool focuses on being **fast, minimal, and developer-friendly**, making it easy to experiment with guitar tab processing, playback, and tooling.

---

## Features

* 🎸 Guitar tablature utilities
* 🎵 MIDI playback support
* ⚡ Lightweight and fast
* 🧩 Developer-friendly architecture
* 🖥 Cross-platform desktop builds
* 🔓 Fully open source

---

## Installation (Development)

Clone the repository:

```bash
git clone https://github.com/JRossouwGitHub/GuitarTabTool.git
cd GuitarTabTool
```

Install dependencies:

```bash
npm install
```

Run the application:

```bash
npm run start
```

---

## Release Builds

Prebuilt binaries are available in the **`release/`** folder.

These builds allow you to run the application **without installing Node.js or project dependencies**.

Typical usage:

1. Download the latest build from the repository releases
2. Extract the archive
3. Run the executable

Example structure:

```
release/
 ├─ GuitarTabTool.exe
 └─ resources/
```
